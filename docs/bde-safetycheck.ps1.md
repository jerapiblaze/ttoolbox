# BitlockerDriveEncryption-SafetyCheck

This script checks for BitLocker-enabled drives, retrieve backup types and drive ids of them.

For convinience, all drives should have recovery key backed up with M$ account. However, if you do not like the idea of giving M$ your backup keys, you should backup the keys somewhere else, typically to a file.

In modern Windows, to enable BitLocker, at least one recovery key backup method must be setup.
