# Documentations

Various snippests and scripts built from my past experiences. They were made to match my own situations. Feel free to create a fork, modify to suit your own use cases.

- `docker` contains many docker compose manifests for creating private services.
- `scripts` contains tons of scripts (mostly bash and powershell) which can help life easier in some way.
- `gists` contains lots of hidden *treasures* (or *trash*, whatever).
